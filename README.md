# texttocpp
Converts text files into C++ programs that print them.

Installation
------------

Read INSTALL.md for installation instructions.

Usage
-----

Returns readable C++ program:

```
  texttocpp [infile]
```

Returns minified C++ program:

```
  texttocpp -m [infile]
```

or

```
  texttocpp --minimal [infile]
```
